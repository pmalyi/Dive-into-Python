Реализація простого класу для читання з файлу

Перше завдання цього тижня — не складне для розігріву. Ваше завдання: написати python-модуль solution.py, всередину якого
необхідно розмістити код класу FileReader. Конструктор цього класу приймає один параметр: шлях до файлу на диску. У класі FileReader
повинен бути реалізований метод read, який повертає рядок - вміст файлу, шлях до якого було вказано під час створення екземпляра класу.
Модуль Python повинен бути написаний таким чином, щоб імпорт класу FileReader з нього не викликав помилок.

При написанні реалізації методу read вам потрібно враховувати випадок, коли при ініціалізації було передано шлях до неіснуючого файлу.
Потрібно обробити виключення FileNotFoundError, що виникає при цьому, і повернути з методу read порожній рядок.

Приклад роботи:

>>> from solution import FileReader
>>> reader = FileReader('not_exist_file.txt')
>>> text = reader.read()
>>> text
''
>>> with open('some_file.txt', 'w') as file:
...     file.write('some text')
...
9
>>> reader = FileReader('some_file.txt')
>>> text = reader.read()
>>> text
'some text'
>>> type(reader)
<class 'solution.FileReader'>
>>>
